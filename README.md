# React Practice App 🚀

Welcome to my first React project! This application was built to practice the fundamental concepts of React, including components, JSX, and project structure, using the lightning-fast **Vite** build tool.

## ✨ Features

- **Component-Based Architecture**: Modularized code using reusable functional components.
- **Dynamic JSX**: Real-time rendering of data, such as the current year, using JavaScript expressions.
- **Custom Header**: A navigation bar with interactive links (Home, About, Contact).
- **Welcome Component**: A styled, reusable welcoming section integrated into the main view.
- **Vite Integration**: High-performance development environment with Hot Module Replacement (HMR).

## 🛠️ Technologies Used

- **React**: For building the user interface.
- **Vite**: Modern frontend build tool.
- **JSX**: JavaScript XML for declarative UI.
- **CSS**: Custom styling for components.

## 🚀 Getting Started

Follow these steps to run the project locally on your machine:

### 1. Clone the repository
```bash
git clone https://github.com/00SnowFlake/react_day1_assignment.git
cd react_day1_assignment
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the development server
```bash
npm run dev
```

The app will be available at `http://localhost:5173`.

## 📂 Project Structure

```text
src/
├── components/
│   ├── Header.jsx   # Custom navigation component
│   └── Welcome.jsx  # Reusable welcome message component
├── assets/          # Static images and icons
├── App.jsx          # Root component
└── main.jsx         # Entry point for the application
```
